# AltiMaster

Real-time token monitoring and trading signal system for memecoins and altcoins in Solana chain. Automatically discovers new tokens via DexScreener, waits for entry confirmations, and tracks take-profit / stop-loss levels.

> **Disclaimer:** This project is currently in active development. Features, APIs, and behavior may change without notice.

---

## How to Use

**This project is still under development, so its use is largely limited.**

1. **Account** – You need an account, which must be created by an administrator.
2. **Funds** – Have funds in a platform that supports live altcoins and memecoins.
3. **Trade** – Buy and sell coins at the same time as the platform indicates.
